# ESG Carbon Accounting Tools

## Overview

This repository contains Python-based tools for corporate greenhouse gas (GHG) accounting.

The first project calculates:

- Scope 1 emissions
- Scope 2 emissions
- Scope 3 emissions
- Total corporate emissions
- Percentage contribution of each scope
- Bar chart visualization

## Business Problem

Organizations need reliable methods to estimate greenhouse gas emissions for ESG reporting, sustainability disclosures, and carbon management.

This project demonstrates a simplified emissions calculator based on the GHG Protocol.

## Technologies

- Python
- Matplotlib

## Features

- Scope 1 calculations
- Scope 2 calculations
- Scope 3 calculations
- Emissions summary
- Percentage analysis
- Automatic chart generation

## How to Run

```bash
python scope3_calculator.py
```
